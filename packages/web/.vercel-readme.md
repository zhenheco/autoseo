# Vercel Monorepo Root Directory

This branch moves the existing Next.js app into `packages/web`.

Before merging and deploying this branch, update the Vercel project manually:

1. Open the Vercel dashboard for the existing 1waySEO project.
2. Go to Project Settings -> General -> Root Directory.
3. Set Root Directory to `packages/web`.
4. Keep production branch and environment variables unchanged.
5. Do not deploy from this migration branch until the PR is reviewed and merged.

`vercel.json` does not support `rootDirectory`, so this must be a dashboard setting.
