# Triage Labels

Canonical label mapping:

- `needs-triage`: maintainer evaluation required.
- `needs-info`: waiting on reporter or customer context.
- `ready-for-agent`: fully specified and suitable for an AFK coding agent.
- `ready-for-human`: needs human product/design/business judgment.
- `wontfix`: will not be actioned.

If a label does not exist in GitHub yet, create it before publishing issues that require it.
