# Issue Tracker

This repo tracks implementation work in GitHub Issues for `zhenheco/autoseo`.

Use `gh issue create`, `gh issue view`, and `gh issue list` from the repo root. Parent PRDs should be created as GitHub issues, and implementation slices should reference the parent issue.

Do not place raw secrets, tokens, private keys, webhook secrets, customer credentials, or full environment values in issue bodies or comments. Use 1Password item names, field names, and platform secret names only.
